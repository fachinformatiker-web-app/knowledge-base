# Algorithmen

<!-- Algorithmen erstellen können

## Pseudocode

-   Informelle Beschreibung von Programmabläufen
-   Verwendung von Texten und Symbolen zur Beschreibung von Programmlogik

## Struktogramm

-   Grafische Darstellung von Programmabläufen
-   Verwendung von symbolischen Elementen zur Beschreibung von Programmlogik

## PAP (Programmablaufplan)

-   Strukturierte Darstellung von Programmabläufen
-   Verwendung von graphischen Symbolen zur Beschreibung von Programmlogik
-   Übersichtliche Darstellung der Programmlogik durch strukturierte Anordnung

## Beispiel-Fragen

-   Was ist Pseudocode und wie wird er verwendet?
-   Wie können Programmabläufe grafisch dargestellt werden?
-   Was ist ein PAP und welche Vorteile hat seine Verwendung bei der Erstellung von Algorithmen?
